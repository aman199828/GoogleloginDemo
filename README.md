
# Google Login Demo




## Run Locally

Clone the project

```bash
  git clone https://github.com/aman199828/googlelogin.git
```


Install dependencies

```bash
npm i

```

Start the server

```bash
node index.js
```

# Then select googlelogin folder
```bash
cd googlelogin

```
Install dependencies

```bash
npm i

```

Start the server

```bash
npm run dev
```
